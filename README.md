### For GNU/Linux Debian 9 Stretch

Clone this repo:

```bash
$ git clone git@github.com:koddr/vdsac.git
```

Run `init.sh` from root dir:

```bash
$ cd vdsac
$ sudo chmod u+x init.sh
$ sudo ./init.sh
```
